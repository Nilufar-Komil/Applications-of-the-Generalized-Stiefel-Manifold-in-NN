# Applications-of-the-Generalized-Stiefel-Manifold-in-NN
I have investigated the use of the Stiefel manifold in neural networks, where weight matrices are constrained to lie on matrix manifolds, and explored various optimization methods for training under these geometric constraints.

# Data set : CIFAR 10
# Architecture: CNN with 4 conv.layers
